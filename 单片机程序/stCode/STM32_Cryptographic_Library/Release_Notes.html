Libraries image format  update History


V2.0.6 / 25- June-2013

Main Changes

  • Seven STM32 Cryptographic Libraries provided for each Development Toolchain:
    • IAR Embedded Workbench for ARM (EWARM) toolchain:
      • M4_CryptoHW_2_0_6.a: First official release of optimized STM32 Cryptographic Library for STM32F41x famillies.
      • M4_CryptoFW_RngHW_2_0_6.a: First official release of STM32 Cryptographic Library Firmware version with  Hardware RNG pheripheral for STM32F4xx famillies.
      • M4_CryptoFW_2_0_6.a: First official release of STM32 Cryptographic Library Firmware version for STM32F40x famillies.
      • M3_CryptoHW_2_0_6.a: New official release of optimized STM32 Cryptographic Library for STM32F21x famillies.
      • M3_CryptoFW_RngHW_2_0_6.a:  First official release of STM32 Cryptographic Library Firmware version with  Hardware RNG pheripheral for STM32F20x famillies.
      • M3_CryptoFW_2_0_6.a: New STM32 Cryptographic Library Firmware version release for STM32F10x and STM32F3xx.
      • M3_CryptoFW_L1xx_2_0_6.a: New STM32 Cryptographic Library Firmware version release for  STM32L1xx famillies.
      • M0_CryptoFW_2_0_6.a: First STM32 Cryptographic Library Firmware version release for STM32F0xx famillies.
    • RealView Microcontroller Development Kit (MDK-ARM) toolchain :
      • M4_CryptoHW_2_0_6.lib: First official release of optimized STM32 Cryptographic Library for STM32F41x famillies.
      • M4_CryptoFW_RngHW_2_0_6.lib: First official release of STM32 Cryptographic Library Firmware version with  Hardware RNG pheripheral for STM32F4xx famillies.
      • M4_CryptoFW_2_0_6.lib: First official release of STM32 Cryptographic Library Firmware version for STM32F40x famillies.
      • M3_CryptoHW_2_0_6.lib: New official release of optimized STM32 Cryptographic Library for STM32F21x famillies.
      • M3_CryptoFW_RngHW_2_0_6.lib:  First official release of STM32 Cryptographic Library Firmware version with  Hardware RNG pheripheral for STM32F20x famillies.
      • M3_CryptoFW_2_0_6.lib: New STM32 Cryptographic Library Firmware version release for STM32F10x and STM32F3xx famillies .
      • M3_CryptoFW_L1xx_2_0_6.lib: New STM32 Cryptographic Library Firmware version release for  STM32L1xx famillies.
      • M0_CryptoFW_2_0_6.lib: First STM32 Cryptographic Library Firmware version release for STM32F0xx famillies.
Note: CRC pheripheral is used by STM32 cryptographic library.

2.0.1 / 14-October-2011

Main Changes

  • Two STM32 Cryptographic Libraries provided for each Development Toolchain:
    • IAR Embedded Workbench for ARM (EWARM) toolchain:
      • STM32_CryptoHW_Lib.a: First official release of optimized STM32 Cryptographic Library for STM32F21x devices .
      • STM32_CryptoFW_Lib.a: New STM32 Cryptographic Library Firmware version release for STM32F10x, STM32L1xx and STM32F2xx famillies.
    • RealView Microcontroller Development Kit (MDK-ARM) toolchain :
      • STM32_CryptoHW_Lib.Lib: First official release of optimized STM32 Cryptographic Library for STM32F21x devices .
      • STM32_CryptoFW_Lib.Lib: New STM32 Cryptographic Library Firmware version release for STM32F10x, STM32L1xx and STM32F2xx famillies.
  • To enable STM32 cryptographic library firmware algorithm, CRC clock shoud be enabled in user application stage.
